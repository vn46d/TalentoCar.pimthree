# 🚗 TalentoCar — Sistema de Gestão para Estética Automotiva
- Sistema web desenvolvido para gerenciamento de agendamentos e clientes de uma estética automotiva, permitindo cadastro, controle de serviços e organização operacional.


## 📌 Funcionalidades
- Cadastro de clientes
- Cadastro de veiculos
- Agendamento de serviços
- Consulta de histórico 
- Painel administrativo 


## 🛠 Tecnologias
- C#
- ASP.NET
- SQL SERVER 
- HTML5
- CSS3
- JAVASCRIPT 
- SQL


## 🏛️ Arquitetura
O sistema segue separação entre:
- Frontend
- Backend (API REST)
- Banco de Dados


## 📍 Objetivo
- Um projeto acadêmico aplicado a um cenário real de operação automotiva, focado em digitalizar e otimizar processos de atendimento e agendamento.


## ▶️ Como rodar
1. Rodar o Program.cs
2. Abrir o util.html
3. Executar o SQL.SERVER
